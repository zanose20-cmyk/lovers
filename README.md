Lovers — تطبيق دردشة صوتية (Skeleton)

محتوى المشروع:
- backend/: خادم Node.js + Express + MongoDB + Socket.IO + AdminJS
- flutter/: تطبيق Flutter (واجهة موبايل)
- admin/: إعداد لوحة إدارة (AdminJS)

تعليمات سريعة (Backend):
1. انسخ ملف .env.example إلى .env واملأ المتغيرات.
2. تشغيل:

```
cd backend
npm install
npm run dev
```

تعليمات سريعة (Flutter):
1. افتح مجلد flutter في IDE (Android Studio / VS Code)
2. تشغيل:
```
flutter pub get
flutter run
```
